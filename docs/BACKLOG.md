# Backlog — идеи и баги от пользователя

Сюда юзер кидает мысли по ходу работы. Опус выныривает → переносит нужное в `sprints/NN/TASKS.md` → отмечает здесь `[→ SNN]`.

Формат записи:
- `[статус] [тег: S01/S02/...] описание`
  - `[ ]` — новое, не разобрано
  - `[→ SNN]` — переехало в спринт N
  - `[x]` — сделано
  - `[~]` — пропущено / не актуально
