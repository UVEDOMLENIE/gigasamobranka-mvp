# Сравнение с существующими решениями

| Критерий | Quizlet | Anki | ChatGPT + ручная работа | ГигаСамобранка |
|---|---|---|---|---|
| **Генерация из PDF/DOCX** | ❌ Нет | ❌ Нет | ⚠️ Ручной копипаст | ✅ Автоматический парсинг + LLM |
| **LLM-генерация на русском** | ❌ Нет | ❌ Нет | ✅ Да, но без структуры | ✅ Структурированный JSON, методические правила |
| **Редактирование карточек** | ⚠️ Только swap | ⚠️ Сложный интерфейс | ❌ Нет (текстовый файл) | ✅ Inline редактор, drag-reorder, autosave |
| **Режим ученика + статистика** | ❌ Только self-test | ❌ Только self-test | ❌ Нет | ✅ Персональная ссылка, сессии, таблица результатов |
| **Печать A4 / DOCX** | ❌ Нет | ❌ Нет | ❌ Ручная вёрстка | ✅ /print/[id] + DOCX экспорт |
| **KaTeX / формулы** | ❌ Нет | ⚠️ Плагины | ⚠️ Ручной ввод LaTeX | ✅ Нативная поддержка |
| **Без регистрации** | ❌ Требует аккаунт | ❌ Требует аккаунт | ✅ | ✅ Owner key в cookie |
| **Стоимость** | Бесплатно / Pro | Бесплатно | Подписка OpenAI | Бесплатно (Vercel Hobby + триал Scarlex) |

## Ключевое отличие

Quizlet и Anki — **хранилища** карточек. Создавать контент учитель должен сам.

ChatGPT — **генератор текста**, но без структуры, без редактора, без доставки ученикам, без статистики.

**ГигаСамобранка** — **end-to-end pipeline**: загрузка → генерация → редактирование → доставка → анализ → печать.
