## Ресурсы MVP

| Ресурс | Что использовали |
|---|---|
| Разработка | 1 AI-agent (Devin Local + Cloud) × 8 дней |
| Хостинг | Vercel Hobby (бесплатно) |
| LLM | Scarlex триальный ключ (Claude Sonnet) |
| БД | SQLite + Vercel Blob (бесплатно, 60KB) |
| Документация | GitHub + Markdown |

## Возможности развития

1. **Mindmap-визуализация** — граф связей между понятиями в наборе
2. **Fork наборов** — учитель копирует чужой набор и адаптирует
3. **Классные группы** — объединение учеников, массовая рассылка
4. **LMS-интеграция** — экспорт в Moodle, Google Classroom
5. **Голосовой ввод** — учитель диктует карточки, Whisper транскрибирует
6. **AI-проверка ответов** — не только "знал/не знал", но и оценка развёрнутого ответа

## Риски и нейтрализация

| Риск | Вероятность | Влияние | Мера |
|---|---|---|---|
| Scarlex rate-limit (429) | Средняя | Высокое | Exponential backoff + mock fallback |
| БД эфемерна на Vercel serverless | Высокая | Высокое | Vercel Blob backup/restore (временно), переход на Turso |
| PPTX парсинг нестабилен | Низкая | Среднее | Fallback на textarea + ручной ввод |
| Мобильная адаптация слабая | Средняя | Низкое | Плеер работает на телефоне, UI не критичен для MVP |
| Утечка API-ключа | Низкая | Высокое | Ключи в env vars, не в коде. Утёчный ключ — временный |
| Триал Scarlex закончится | Средняя | Среднее | Fallback на GigaChat или mock |
