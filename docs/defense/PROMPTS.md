# Промпты и интеграция с LLM

## Системный промпт (SYSTEM_PROMPT)

```text
Ты опытный методист-составитель учебных карточек. Твоя задача — создать качественные карточки для школьников.

ПРАВИЛА:
- Вопрос должен быть конкретным, проверяющим понимание материала
- Ответ — краткий, точный, информативный (1–3 предложения)
- Не повторяй вопрос в ответе
- Используй терминологию из исходного текста
- difficulty: easy (определения, факты), medium (применение, связи), hard (анализ, синтез)
- source — имя файла-источника

Формат ответа — ТОЛЬКО JSON-массив, без пояснений:
[{"question": "...", "answer": "...", "source": "...", "difficulty": "..."}]
```

## Пользовательский промпт (buildPrompt)

```text
Предмет: {subject}
Класс: {grade}
Тема: {topic}
Количество карточек: {count}
Сложность: {difficulty}

Исходные материалы:
---
{sources[0].text} (truncated to 4000 chars)
---

Создай {count} карточек.
```

## Почему такой промпт

1. **Контекст предмета и класса** — LLM знает уровень сложности (1 класс vs 11).
2. **Тема** — фокус на конкретной области, не всё подряд.
3. **Уровни сложности** — easy/medium/hard позволяют адаптировать под разные группы учеников.
4. **Source** — методист видит откуда взялся вопрос (для проверки релевантности).
5. **JSON-only** — парсинг детерминированный, без markdown-артефактов.
6. **Контекст 4000 символов** — баланс между качеством и стоимостью токенов.

## Интеграция Scarlex (Claude Sonnet 4.6)

- **API**: OpenAI-compatible (`https://api.scarlex.ru/v1`)
- **Модель**: `claude-sonnet-4.6` (умнее haiku, дешевле opus)
- **Retry**: exponential backoff 1s → 3s → 9s на 429
- **Fallback**: если API падает → mock (regex-парсер текста)

## Интеграция GigaChat

- **OAuth**: Sber OAuth 2.0 (`ngw.devices.sberbank.ru`)
- **Модель**: `GigaChat` (стандартная)
- **Temperature**: 0.4 (консервативно, меньше галлюцинаций)
- **Max tokens**: 4096 (достаточно для 20–30 карточек)

## Rate-limiting

- `LLM_REQUESTS_PER_HOUR=30` на пользователя (по `owner_key`)
- Проверка перед каждым запросом к LLM
- Если лимит исчерпан — ответ 429 с указанием времени до сброса
