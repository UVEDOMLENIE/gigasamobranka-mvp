Проблема и пользователи

Проблема

  • 2–3 часа на ручное составление карточек
  • 30+ файлов в «папке учителя» — хаос
  • Учитель тратит время на рутину вместо детей

Целевая аудитория

  • Учителя 1–11 класса
  • Методисты, репетиторы
  • Школы РФ, без бюджета на EdTech

Решение

Загрузка материала → LLM → карточки → редактирование → ссылка ученикам → результаты → печать

Сценарии

  • Загрузить PDF/DOCX/TXT
  • Получить набор за 30 сек
  • Править inline, drag-reorder
  • Дать ссылку ученику (без регистрации)
  • Смотреть результаты сессий
  • Печать A4 или DOCX экспорт

  • БД эфемерна → Vercel Blob backup
  • PPTX не парсится → fallback textarea
  • Триал закончится → GigaChat / mock
