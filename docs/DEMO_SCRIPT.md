# Демо-сценарий для проверки

## Цель

Пройти весь flow за 3–5 минут и убедиться, что ничего не падает.

## Шаги

### 1. Главная
- Открыть `/`.
- Убедиться, что видны 5 демо-ссылок.
- Убедиться, что есть форма «Составить свой набор».

### 2. Демо-набор
- Кликнуть «Словарные слова: Зима» (первая ссылка).
- Должен открыться редактор `/sets/...` с 8 карточками.

### 3. Редактор
- Проверить, что карточки отображаются (вопрос + ответ).
- Отредактировать первую карточку.
- Подождать 2 секунды, убедиться в «✓ Сохранено».

### 4. Плеер учителя
- Кликнуть «▶ Открыть плеер».
- Должен открыться `/play/...?mode=teacher`.
- Увидеть вопрос.
- Кликнуть на карточку — должна перевернуться.
- Увидеть «✓ Знал» / «✗ Не знал».
- Кликнуть «✓ Знал».

### 5. Ссылка ученикам
- В редакторе кликнуть «🔗 Ссылка ученикам».
- Должна скопироваться ссылка или появиться prompt с URL.

### 6. Ученик
- В новой вкладке открыть скопированную ссылку `/play/...` (без `?mode=teacher`).
- Ввести имя «Тест».
- Кликнуть «Начать ▶».
- Пройти 2–3 карточки (перевернуть + Знал/Не знал).

### 7. Результаты
- Вернуться к редактору.
- Кликнуть «📊 Результаты».
- Убедиться, что «Тест» виден в таблице.

### 8. Печать
- Кликнуть «🖨 Печать».
- Убедиться, что страница `/print/...` загрузилась с карточками.

## Ожидаемый результат

Все 8 шагов проходят без ошибок. Если где-то падает — это blocker.
