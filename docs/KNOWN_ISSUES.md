# Известные проблемы

## Баги

1. **navigator.clipboard на HTTP/IP**
   - Где: кнопка «🔗 Ссылка ученикам» в редакторе.
   - Почему: `navigator.clipboard.writeText()` требует HTTPS или localhost.
   - Решение: уже есть fallback через textarea + `document.execCommand("copy")`.
   - Статус: пофикшено, но стоит проверить на деплое.

2. **Парсинг DOCX иногда падает**
   - Где: загрузка файлов `.docx`.
   - Почему: некоторые DOCX не валидны как zip.
   - Решение: warning + fallback на textarea; генерация идёт, если есть текст.
   - Статус: пофикшено мягко.

3. **Страница «Сгенерировать» может сабмитить форму как GET**
   - Где: форма на главной.
   - Почему: `form` без `e.preventDefault()` или старая вкладка.
   - Решение: `e.preventDefault()` добавлено; `method="post" action="#" noValidate`.
   - Статус: пофикшено, проверить на чистом браузере.

## Технический долг

- Нет тестов в самом проекте (Playwright-скрипты лежат вне репы).
- Нет CI/CD.
- `.env.local` может содержать ключи — не коммитить.
- `local.db` в `.gitignore`.

## Риски перед сдачей

- Dev server может зависнуть (было на 3000, стартовали на новом порту).
- Chrome/Edge CDP может не отвечать.
- Рекомендация: деплой на Vercel/Render + smoke-тест.
