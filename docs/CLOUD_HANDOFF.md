# Devin Cloud Handoff

## Миссия

Задеплой MVP на живой URL. Проверь, что демо-сценарий проходится. Исправь только blocker-баги. Не расширяй scope.

## Контекст

- Хакатон СберОбразование, Кейс 2: Генератор учебных карточек.
- Срок: через 1–2 дня.
- Команда: UVEDOMLENIE / GigaSamobranka Team.
- Репа: `https://github.com/UVEDOMLENIE/gigasamobranka-mvp`

## Что уже работает (не ломай)

- Главная страница с формой генерации.
- Демо-ссылки `/api/demo?id=...` → серверный редирект → редактор.
- Редактор карточек: правка, добавление, удаление, сохранение.
- Плеер учителя: flip, next/prev, Знал/Не знал, fullscreen, клавиши.
- Ученик: вход по ссылке `/play/[id]`, имя без регистрации, прохождение.
- Результаты: `/sets/[id]/results` таблица сессий.
- Печать: `/print/[id]` A4-страница.
- KaTeX для формул.
- Mock-генерация fallback (без расхода API).
- Scarlex V2 / OpenAI-compatible + GigaChat в `/settings`.

## Стек

- Next.js 16.2.6, React 19, Tailwind v4, TypeScript.
- DB: Drizzle + `@libsql/client` (SQLite локально).
- LLM: mock-first; реальный через `/settings` или env.
- npm.

## Как запустить

```bash
npm install
npm run db:migrate   # создать local.db
npm run dev          # localhost:3000
# или
npm run dev:lan      # 0.0.0.0:3000 для локалки
```

## Демо-сценарий (проверь после деплоя)

1. Открыть `/`.
2. Кликнуть демо «Словарные слова: Зима».
3. Убедиться, что открылся редактор с карточками.
4. Открыть плеер, перевести карточку.
5. Скопировать ссылку ученикам.
6. В другой вкладке открыть ссылку, ввести имя, пройти 2–3 карточки.
7. Вернуться к результатам, увидеть прохождение.
8. Открыть печать, убедиться, что страница загружается.

## Правила работы (жёстко)

1. Каждый логический шаг — commit. Сообщение на русском или английском.
2. Не добавляй новые крупные фичи (mindmap, регистрация, RAG).
3. Не меняй дизайн радикально. Только polish: отступы, размер шрифта, цвета.
4. Не храни секреты в коде. API-ключи только через `/settings` или env.
5. Не используй real names в коммитах, README, комментариях.
6. Перед каждым изменением — `npm run lint && npx tsc --noEmit`.
7. Если не знаешь — спроси, не гадай.

## Приоритеты правок
