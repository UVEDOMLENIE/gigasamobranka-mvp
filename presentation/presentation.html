Проблема

  • • Ручное форматирование формул
  • • Нет времени на индивидуализацию

  • • Скучные однообразные задания

👨‍🏫 Для учителя

  1. 1. Открой демо "Словарные слова: Зима"
  2. 2. Посмотри редактор карточек
  3. 3. Открой плеер (кнопка ▶)
  4. 4. Скопируй ссылку ученикам
  5. 5. Проверь результаты

  1. 1. Открой ссылку от учителя
  2. 2. Введи имя
  3. 3. Проходи карточки
  4. 4. Переворачивай кликом
  5. 5. Отмечай знание
